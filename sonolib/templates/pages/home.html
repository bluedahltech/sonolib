Welcome to Sonolib!

This is a public rest api for Sono.ly. It is a curated library of sounds, wavetables, impulse responses, and sound kits.

If you wish to contribute sounds to this library, send an email to darren@bluedahl.tech and we can get you set up with an account.

Otherwise, feel free to use the public rest api however you wish. See API doc here.
